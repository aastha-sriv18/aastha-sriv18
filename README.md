## 🚀 Featured Projects

###  [ColorCense](https://github.com/aastha-sriv18/ColorCense)
> AI-powered accessibility application designed to assist individuals with color blindness through real-time color interpretation and visual enhancements.

-  Published on the **Google Play Store** and actively used by users across multiple countries
-  Built real-time color detection and identification features for improved accessibility
-  Developed image conversion tools to generate color-blind-friendly visualizations
-  Integrated **Google ML Kit OCR** for extracting and enhancing readability of text from images and documents
-  Designed specialized support for marine biology color-based assessments
-  Open-source project with a privacy-first approach and **zero user data collection**



###  [AI-Powered Store Intelligence System](https://github.com/aastha-sriv18/Store-Intelligence-System)
> End-to-end retail analytics platform developed for the **Purplle Tech Challenge 2026**, transforming CCTV footage into actionable business intelligence.

-  Leveraged **Computer Vision** and **Multi-Object Tracking** to analyze customer movement patterns
-  Generated insights on zone popularity, dwell time, footfall trends, and store utilization
-  Built an event-processing pipeline to convert raw video streams into structured analytics
-  Designed interactive dashboards for visualizing customer behavior and retail performance
-  Enabled data-driven decision-making through real-time store intelligence



###  [Plant Disease Detection & Recommendation System](https://github.com/aastha-sriv18/AI-Driven-Web-Application-for-Automated-Disease-Detection-in-Rice-and-Pulse-Crops)
> AI-driven agricultural solution that detects plant diseases and provides actionable recommendations using Deep Learning and Generative AI.

-  Trained an **EfficientNet-based model** capable of classifying **40+ plant diseases**
-  Integrated the model into a full-stack web application for real-time disease prediction
-  Utilized **Google Gemini API** to generate AI-powered disease reports and treatment suggestions
-  Implemented secure authentication and scan-history management using SQL databases
-  Designed an intuitive workflow for uploading plant images and receiving instant diagnosis
